# Search App

## Requirements

1. Yarn >= v1.22.15
2. Node >= v16.13.2

## Setup

1. `yarn install`

2. Ensure the API is started and ready - see [here](https://github.com/saadsharif/ttds-group/api). Note, we currently assume port 5000.


## Running

The following will start the UI on port 3000.

`yarn start`
